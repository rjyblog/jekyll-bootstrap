**普通字符替换**
```
param_name=ro.build.id
sed -i "s/${param_name}=.*/${param_name}=new_value/g" build.prop
```
本例用于修改build.prop文件中的ro.build.id参数的值

**二进制内容替换**
```
sed -i 's/\x00/\x0a/g' %s      #本例是把'\0'替换为'\n'
```

**删除文件中的某一行**
```
sed -i \'1d\' %s   #删除第一行的文字
```
